# PostgreSQL-Backup

Two bash scripts to Dump/Restore 💾💿 a PostgreSQL-Database within a Docker-Container 🐋


# Usage

**Download the File/s**

wget <https://github.com/>  

**Give Executive permissions**

chmod +x <file.sh>

**Execute Script with Root**

./<file.sh>
